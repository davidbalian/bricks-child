# AutoAgora SEO Growth Baseline

Created: 2026-05-22

## Purpose

This document records the first SEO growth baseline for AutoAgora using Google Search Console, GA4, and listing inventory exports. It should be used as the starting point for the SEO plan and future monthly comparisons.

## Data Sources

- Google Search Console exports, last 28 days:
  - Global queries
  - Global pages
  - Devices
  - Countries
  - Search appearance
  - Page-filtered query exports for key URLs
- GA4 exports, last 28 days:
  - Landing pages
  - Pages and screens
- AutoAgora inventory export:
  - `autoagora-seo-inventory-2026-05-22-200831.csv`

## Search Console Baseline

Overall GSC performance, last 28 days:

- Clicks: 1,010
- Impressions: 41,200
- Average CTR: 2.4%
- Average position: 10.7

Main observations:

- Homepage currently carries most generic marketplace search demand.
- `/cars/` has meaningful impressions but weak CTR and rankings.
- Some make/model pages already show clear buyer-intent demand.
- Some individual listings and dealer pages are ranking, which confirms marketplace inventory can generate organic traffic.

## Priority Page Findings

### `/cars/`

Observed query signals:

- `used cars cyprus`
- `cars for sale in cyprus`
- `where to buy used cars`
- `range rover for sale cyprus`
- `mercedes for sale cyprus`

Issue:

- `/cars/` is the correct marketplace destination, but it underperforms in organic search.

Why it matters:

Priority:

## Data Quality Issues

### City Field Missing

The inventory export city column is blank for active listings.

Possible causes:

- Export is reading the wrong meta key.
- City is stored under another key such as `car_district`, location picker fields, or ACF formatted data.
- Existing listings may have incomplete location data.

Why it matters:

- City SEO pages for Nicosia, Limassol, Larnaca, and Paphos cannot be properly evaluated without city-level inventory data.

Priority:

- High

Expected impact:

- Better city landing analysis and more accurate local SEO prioritization.

## Immediate SEO Plan Inputs

The SEO plan should prioritize:

1. Strengthening `/cars/` as the main non-brand marketplace landing page.
2. Improving CTR and copy for Toyota Yaris Cross, Volkswagen Golf, and selected high-intent make/model pages.
3. Creating a stale inventory workflow for dealers.
4. Auditing high-view zero-contact listings.
5. Fixing city data export and city inventory analysis.
6. Improving dealer reporting around freshness, exposure, and contacts.
7. Expanding managed SEO landing pages only where search demand and active supply overlap.

## Current Bottleneck

The main SEO bottleneck is not keyword discovery.

The main bottleneck is marketplace quality:

- Inventory is too stale.
- Contact conversion is too low.
- City data is incomplete.
- Too many listings receive views without seller contact.

SEO growth should be tied to inventory freshness, dealer accountability, and page-level search intent matching.
